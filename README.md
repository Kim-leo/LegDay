## 프로젝트 소개
1. 진행 기간 및 규모: 2023.12 ~ 2023.03 (4개월) / 개발 1명 (본인)
2. 사용 기술
- 개발도구: XCode (최소 사양 iOS 15.0)
- 사용언어: Swift
- 아키텍처: MVVM 채택
3. 일정관리: Notion
